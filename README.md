# Logan Böttger - Maschinenbau Portfolio

Einseitige Portfolio-Website (One-Pager) für die Bewerbung um einen dualen Studienplatz im Maschinenbau.

## Funktionen & Besonderheiten
- **Interaktiver Cursor-Reveal-Effekt (Hero-Bereich)**: 3D-CAD-Würfel blendet sich sanft im Bereich des Mauszeigers frei.
- **Scroll-Text-Animation**: Textpassagen und Headlines bauen sich wortweise beim Scrollen auf.
- **Sektionsübergänge**: Fließendes Ein- und Ausblenden der Hauptabschnitte mit Skalierungseffekten.
- **Intro-Ladeanimation**: Eleganter Ladebildschirm beim Aufrufen der Seite.
- **100% Statisch & GitHub Pages Kompatibel**: Keine serverseitigen Abhängigkeiten, rein relative Pfade, optimiert für schnelles Laden.

## Struktur für GitHub Pages Hosting
- `index.html`: Einstiegspunkt
- `.nojekyll`: Verhindert Jekyll-Verarbeitung
- `/assets/css/style.css`: Stylesheet
- `/assets/js/script.js` & `/assets/js/cursor-reveal.js`: JavaScript-Module
- `/assets/images/`: Fotos, Grafiken & Icons
